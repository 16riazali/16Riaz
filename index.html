☁️ Cloud Sync Setup

1

Supabase Account খোলো (ফ্রি)

নিচের লিংকে যাও, Google দিয়ে signup করো:

https://supabase.com
2

New Project তৈরি করো

"New Project" → Project name: d90-riaz → Region: Southeast Asia → একটি password দাও → Create।

3

Table তৈরি করো (SQL Editor)

Left sidebar → SQL Editor → "New Query" → নিচের কোডটা paste করো → Run:

CREATE TABLE d90_state ( id TEXT PRIMARY KEY DEFAULT 'riaz', data JSONB NOT NULL DEFAULT '{}', updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE d90_state ENABLE ROW LEVEL SECURITY; CREATE POLICY "open" ON d90_state FOR ALL USING (true) WITH CHECK (true);
4

API Keys নাও

Left sidebar → Project Settings → API → দুটো জিনিস copy করো:

① Project URL
② anon public key

5

নিচে paste করো ও Save করো

6

Netlify-তে Host করো (ফ্রি)

① netlify.com → Free account খোলো
② Dashboard → "Add new site" → "Deploy manually"
③ HTML ফাইলটা drag & drop করো
④ তোমার URL পেয়ে যাবে! (যেমন: riaz-90days.netlify.app)

https://netlify.com
